Utility Classes : Spacing

Utility : Spacing

I. Description

Collection of , used to define an element's spacing.

Structured on 3 pillars: padding | margin | border

II. Initialization

Download and refference the [ asterisk-utility-spacing.css ] CSS file.

* Not required if already using the [ asterisk-utility.css ] main CSS file.

<link href="--myPath--/asterisk-utility-spacing.css" type="text/css" rel="stylesheet">

III. Usage

Add any of the provided classes to the Element.

Format : <div class="*propertyRefference*-*size*"> </div>
Example : <div class="m-0"> </div> // sets margin: 0
Example : <div class="pl-0"> </div> // sets padding-left: 0
Example : <div class="bt-0"> </div> // sets border-top: 0

IV. Class-name Syntax

Each className has 2 parts : propertyRefference and size .

V. Responsive Utilities : Viewport-Prefixes

To create different scenarios for specific viewports, any class can be prefixed with any *viewport*- or *viewport*-o- syntax.

Format : <div class="*viewportPrefix*-className"> </div>
Example : <div class="m-0"> </div> // sets margin: 0 on all viewports
Example : <div class="sm-m-0"> </div> // sets margin: 0 on small(sm) viewports or larger
Example : <div class="sm-o-m-0"> </div> // sets margin: 0 on small(sm) viewports only

VI. Changing default sizes

Each [size-value] follows the exemplified model:

Each [size-value] inherits from the --astx-util-sp-*sizeValue* CSS variable, set on the :root pseudo-element.

* abreviated from asterisk-utility-spacing-*sizeValue*

Each [size-value] is provided with fallbacks to a default value:
* if CSS variables are not supported;
* if the varible is not defined.

:root { --astx-util-sp-xs: 10px; } .m-xs { margin: 10px !important; margin: var(--astx-util-sp-xs, 10px) !important; }
Any [size-value] can be modified :
for all Elements
by changing the value of the [ --astx-util-sp-*sizeValue* ] CSS variable, on the [body].
for specific Elements
by defining the [ --astx-util-sp-*sizeValue* ] CSS variable on a Parent, or on the Element itself.
<div class="myDiv-1 mv-xs"> </div> body { --astx-util-sp-xs: 5px; } // Will modify the value for both divs
<div class="myDiv-2 mv-xs"> </div> .myDiv-1 { --astx-util-sp-xs: 5px; } // Will modify the value for .myDiv-1 only
Under development: [*viewport*-mh-auto] class (left|right margin:auto, for centering block elements), following the same model.
