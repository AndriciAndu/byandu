Grid System : Row

Utility : Grid System - Row

I. Description

Collection of defining a Grid System styling for setting equal-width first-level children.

II. Initialization

Download and refference the [ asterisk-utility-gridSystem-row.css ] CSS file.

<link href="--myPath--/asterisk-utility-gridSystem-row.css" type="text/css" rel="stylesheet">

Alternatively, a {*}-spec.css file can be generated within the [Code] Tab

<link href="--myPath--/asterisk-utility-gridSystem-row-spec.css" type="text/css" rel="stylesheet">

III. Usage

Classes are applied to the HTML Element and take effect on the Element's first-level children.

The row-n classes (placed on the Parent Element) offers an alternative to setting multiple col-m classes on the first-level Children Elements, when working with equal-width items.

The row-equal class will place all first-level Children on a single row, and will make them have the same width.

It will take effect only if there are a maximum of 12 first-level Children.

Both examples achieve the same result : 

										<div class="row-2">
										    <div> </div>
										    <div> </div>
										</div>
										
										<div>
										    <div class="col-6"> </div>
										    <div class="col-6"> </div>
										</div>
										
										<div>
										    <div class="col-6"> </div>
										    <div class="col-6"> </div>
										</div>

<div class="row-equal">

<div> </div>

<div> </div>

</div

The Row Grid-System CSS declarations have a higher specificity than those of the 12-Col Grid-System CSS declarations.

												    12-Col Grid System - CSS Rule Specificity
												    .col-n { }

												    Row Grid System - CSS Rule Specificity
												    .row-n  > * { }
												    .row-equal > *:first-child:nth-last-child(n) ~ * { }

Row Grid-System row-n classes set on the Parent will override any 12-Col Grid-System col-n classes set on the First-level Children . 

													 <!-- all children will have width: 25% --> 
												    <div class="row-4">
												        <div class="col-6"> </div>
												        <div class="col-6"> </div>
												    </div>

													 <!-- all children will have equal-width --> 
												    <div class="row-equal">
												        <div class="col-3"> </div>
												        <div class="col-9"> </div>
												    </div>

V. Responsive Utilities : Viewport-Prefixes

To create different scenarios for specific viewports, any class can be prefixed with any *viewport*- or *viewport*-o- syntax.

Format :
<div class="{viewportPrefix}-{className}"> </div>
Example :
<div class="row-4"> </div>
// sets witdh: 25% for children on all viewports
Example :
<div class="sm-row-4"> </div>
// sets witdh: 25% for children on small(sm) viewports or larger
Example :
<div class="sm-o-row-4"> </div>
// sets witdh: 25% for children on small(sm) viewports only
