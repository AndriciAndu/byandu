Utility Classes : Spacing - Row

Utility : Spacing - Row

I. Description

Collection of , used to define the spacing for an Element's first-level Children.

Structured on 3 pillars: padding | margin | border

II. Initialization

Download and refference the [ asterisk-utility-spacing-row.css ] CSS file.

<link href="--myPath--/asterisk-utility-spacing-row.css" type="text/css" rel="stylesheet">

Alternatively, a {*}-spec.css file can be generated within the [Code] Tab

<link href="--myPath--/asterisk-utility-spacing-row-spec.css" type="text/css" rel="stylesheet">

III. Usage

Classes are applied to the HTML Element and take effect on the Element's first-level children.

Format :
<div class="row-{propertyRefference}-{size}"> </div>
Example :
<div class="row-m-0"> </div>
// sets margin: 0 on first-level Children
Example :
<div class="row-pl-0"> </div>
// sets padding-left: 0 on first-level Children
Example :
<div class="row-bt-0"> </div>
// sets border-top: 0 on first-level Children

The utility-spacing-row doesn't have the !important declaration within it's syntax (contrary to the utility-spacing syntax)

As a result, any utility-spacing-row rules (set on the Parent) can be overriden for specific children, by using the utility-spacing syntax.

<div class="row-m-md"> <-- all children will have margin: 15px --> <div> </div> <div> </div> <div class="m-0"> </div> <-- this child will have margin: 0 --> </div>

V. Responsive Utilities : Viewport-Prefixes

To create different scenarios for specific viewports, any class can be prefixed with any *viewport*- or *viewport*-o- syntax.

Format :
<div class="{viewportPrefix}-{className}"> </div>
Example :
<div class="row-m-0"> </div>
// sets margin: 0 for Children on all viewports
Example :
<div class="sm-row-m-0"> </div>
// sets margin: 0 for Children on small(sm) viewports or larger
Example :
<div class="sm-o-row-m-0"> </div>
// sets margin: 0 for Children on small(sm) viewports only
