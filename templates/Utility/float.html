Utility Classes : Float

Utility : Float

I. Description

Collection of used to define an element's float and/or clear properties.

II. Initialization

Download and refference the [ asterisk-utility-float.css ] CSS file.

<link href="--myPath--/asterisk-utility-float.css" type="text/css" rel="stylesheet">

Alternatively, a {*}-spec.css file can be generated within the [Code] Tab

<link href="--myPath--/asterisk-utility-float-spec.css" type="text/css" rel="stylesheet">

III. Usage

Classes are applied to the HTML Element and take effect on the HTML Element itself.

Format :
<div class="float-{value}"> </div> Format :
<div class="clear-{value}"> </div>
Example :
<div class="float-left"> </div>
// sets float: left
Example :
<div class="clear-left"> </div>
// sets clear: left

IV. Class-name Syntax

Each className follows the format : float-{ value } or clear-{ value }.

Class Effect
float-left float: left
float-right float: right
float-none float: none
clear-left clear: left
clear-right clear: right
clear-both clear: both
float-clear (extra class) clear: both

V. Responsive Utilities : Viewport-Prefixes

To create different scenarios for specific viewports, any class can be prefixed with any *viewport*- or *viewport*-o- syntax.

Format :
<div class="{viewportPrefix}-{className}"> </div>
Example :
<div class="float-left"> </div>
// sets float: left on all viewports
Example :
<div class="sm-float-left"> </div>
// sets float: left on small(sm) viewports or larger
Example :
<div class="sm-o-float-left"> </div>
// sets float: left on small(sm) viewports only
