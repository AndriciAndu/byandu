Utility Classes : 12 column grid system

Utility : Grid System - 12 Col

I. Description

Collection of defining a Responsive 12-Column Grid System.

II. Initialization

Download and refference the asterisk-utility-gridSystem-12col.css CSS file.

<link href="--myPath--/asterisk-utility-gridSystem-12col.css" type="text/css" rel="stylesheet">

Alternatively, a {*}-spec.css file can be generated within the [Code] Tab

<link href="--myPath--/asterisk-utility-gridSystem-12col-spec.css" type="text/css" rel="stylesheet">

III. Usage

Classes are applied to the HTML Element and take effect on the HTML Element itself.

Format :
<div class="{propertyRefference}-{size}"> </div>
Example :
<div class="col-3"> </div>
// sets width: 25% (3 out of 12)
Example :
<div class="col-pull-9"> </div>
// sets left: 75% (9 out of 12)
Example :
<div class="col-push-12"> </div>
// sets right: 100% (12 out of 12)
Example :
<div class="col-offset-6"> </div>
// sets margin-left: 50% (6 out of 12)

V. Responsive Utilities : Viewport-Prefixes

To create different scenarios for specific viewports, any class can be prefixed with any *viewport*- or *viewport*-o- syntax.

Format :
<div class="{viewportPrefix}-{className}"> </div>
Example :
<div class="col-6"> </div>
// sets witdh: 50% on all viewports
Example :
<div class="sm-col-6"> </div>
// sets witdh: 50% on small(sm) viewports or larger
Example :
<div class="sm-o-col-6"> </div>
// sets witdh: 50% on small(sm) viewports only
