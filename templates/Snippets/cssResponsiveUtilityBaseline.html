Snippets : CSS Resp.Responsive Syntax

Baseline Syntax : set-up responsive utility-classes, across different viewports.

The xs- prefix is not included, as the unprefixed class applies the same effect.

The xl-o- prefix is not included, as any declaration with xl- will apply to only xl screens by design.

					    /* All viewports */
					        .myClass { }

					    /* xs-only */
					    @media (max-width: 575.98px) {
					        .xs-o-myClass { }
					    }

					    /* sm */
					    @media (min-width: 575.98px) {
					        .sm-myClass { }
					    }

					    /* sm-only */
					    @media (min-width: 575.98px) and (max-width: 767.98px) {
					        .sm-o-myClass { }
					    }

					    /* md */
					    @media (min-width: 767.98px) {
					        .md-myClass { }
					    }

					    /* md-only */
					    @media (min-width: 767.98px) and (max-width: 991.98px) {
					        .md-o-myClass { }
					    }

					    /* lg */
					    @media (min-width: 991.98px) {
					        .lg-myClass { }
					    }

					    /* lg-only */
					    @media (min-width: 991.98px) and (max-width: 1199.98px) {
					        .lg-o-myClass { }
					    }

					    /* xl  */
					    @media (min-width: 1199.98px) {
					        .xl-myClass { }
					    }
