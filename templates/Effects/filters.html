Tap item : Show Code

Click item : Show Code | Hover item : Remove Filter

I. Description

Collection of which apply filter overlay effects - tipically used for image elements.

This collection was taken from Yan Zhu's Instagram.css library. Send all merits his way.

Note: The hasFilter-hvr and hasFilter-hvr-remove scenarios are not available in the original library.

II. Initialization

Download and refference the asterisk-effects-filters.css CSS file. CSS file.

<link href="--myPath--/asterisk-effects-filters.css" type="text/css" rel="stylesheet">

Alternatively, a {*}-spec.css file can be generated within the [Code] Tab

<link href="--myPath--/asterisk-effects-filters-spec.css" type="text/css" rel="stylesheet">

III. Usage

Classes can be applied to the image directly, or to any of its' Parent Elements.

(1) hasFilter-{value} classes | filter is always applied

<img class="hasFilter-valencia" src="insertImgSrc">

(2) hasFilter-hvr-{value} classes | filter is applied on hover

<img class="hasFilter-hvr-valencia" src="insertImgSrc">

(3) hasFilter-{value} classes + hasFilter-hvr-remove class | filter is applied by default and is removed when hovered

<img class="hasFilter-valencia hasFilter-hvr-remove" src="insertImgSrc">
