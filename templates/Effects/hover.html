I. Description

Collection of which apply hover effects - tipically used for button and span elements.

II. Initialization

Download and refference the asterisk-effects-hover.css CSS file. CSS file.

<link href="--myPath--/asterisk-effects-hover.css" type="text/css" rel="stylesheet">

Alternatively, a {*}-spec.css file can be generated within the [Code] Tab

<link href="--myPath--/asterisk-effects-hover-spec.css" type="text/css" rel="stylesheet">

III. Usage

Classes are applied to the HTML Element and take effect on the HTML Element itself.

Format : <div class="hvr-{value}"> </div>
Example : <div class="hvr-forward"> </div>

IV. Available Classes

Group Effect : All Effects move the Element with the transform property.

Requirements : The HTML Element to which the class/effect is applied to, must not have other transform declarations.

* If other transform declarations are required, consider placing them on an .

<div class="myUtilityParent">
<div class="hvr-grow"></div>
</div>

/* place transform declarations here */
}
