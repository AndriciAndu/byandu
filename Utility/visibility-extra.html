Utility Classes : Visibility - Extra Utilities

Utility : Visibility - Extra

Collection of , used to define an element's visibility / interactivity.

Structured on 6 pillars:

before-none (hide :before pseudo-element) | after-none (hide :after pseudo-element)

invisible (visibility) | transparent (opacity) | unselectable (user-select) | unclickable (pointer-events)

Download and refference the asterisk-utility-visibility-extra.css CSS file.

<link href="--myPath--/asterisk-utility-visibility-extra.css" type="text/css" rel="stylesheet">

Alternatively, a {*}-spec.css file can be generated within the [Code] Tab

<link href="--myPath--/asterisk-utility-visibility-extra-spec.css" type="text/css" rel="stylesheet">

* If only the baseline, non-responsive classes are desired, the following snippets can be copied to your CSS file. 

																	    .invisible                  { visibility:     hidden !important; } 
																	    .transparent                { opacity:        0      !important; } 		
																	    .unselectable               { user-select:    none   !important; } 
																	    .unclickable, .pointer-none { pointer-events: none   !important; }

* If only the {psuedoElement}-none classes are desired, the following snippets can be copied to your CSS file. 

																	    .before-none:before , 
																	    .after-none:after   { display: none !important; }

																	    @media (max-width: 575.98px)                            { .xs-before-none:before , .xs-after:after { display: none !important; } }
																	    @media (min-width: 575.98px) and (max-width: 767.98px)  { .sm-before-none:before , .sm-after:after { display: none !important; } }
																	    @media (min-width: 767.98px) and (max-width: 991.98px)  { .md-before-none:before , .md-after:after { display: none !important; } }
																	    @media (min-width: 991.98px) and (max-width: 1199.98px) { .lg-before-none:before , .lg-after:after { display: none !important; } }
																	    @media (min-width: 1199.98px)                           { .xl-before-none:before , .xl-after:after { display: none !important; } }

Classes are applied to the HTML Element and take effect on the HTML Element itself.

Example :
<div class="invisible"> </div>
// sets visibility: hidden;
Example :
<div class="transparent"> </div>
// sets opacity: 0;
Example :
<div class="unselectable"> </div>
// sets user-select: none;
Example :
<div class="unclickable"> </div>
// sets pointer-events: none;

To create different scenarios for specific viewports, any class can be prefixed with any *viewport*- syntax.

Format : <div class="{viewportPrefix}-{className}"> </div>
Example :
<div class="transparent"> </div>
// sets opacity: 0; on all viewports
Example :
<div class="sm-transparent"> </div>
// sets opacity: 0; on small(sm) viewports only
Example :
<div class="md-transparent"> </div>
// sets opacity: 0; on medium(md) viewports only
