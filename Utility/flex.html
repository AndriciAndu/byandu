Utility : Flex

Collection of , used to define an element's flex properties.

Download and refference the asterisk-utility-flex.css CSS file.

<link href="--myPath--/asterisk-utility-flex.css" type="text/css" rel="stylesheet">

Alternatively, a {*}-spec.css file can be generated within the [Code] Tab

<link href="--myPath--/asterisk-utility-flex-spec.css" type="text/css" rel="stylesheet">
There are two categories of classes available :

* Parent Classes - applied on the Parent Element to affect it's first-level Child Elements.

Format :
<div class="flex-{value}"> </div> Format :
<div class="flex-{axis}-{value}"> </div>
Example :
<div class="flex-wrap"> </div>
// will allow wrap to occur
Example :
<div class="flex-main-center"> </div>
// will center the content along the main-axis

* Child Classes - applied on a Child Element to override the style set by the Parent Element

Format :
<div class="flex-self-{value}"> </div>
Example :
<div class="flex-self-end"> </div>
// will align the child element at the end of the cross-axis

Classes applied on Parent Elements follow the format : flex-{ value } or flex-{ axis }-{ value }.

Sets the main axis and it's direction - how the child-elements (items) will flow

Class Style
flex-row flex-direction: row;
flex-col flex-direction: column;
flex-row-rev flex-direction: row-reverse;
flex-col-rev flex-direction: column-reverse;

Sets if the child-elements will wrap to multiple lines, along the main-axis

Class Style
flex-wrap flex-wrap: wrap;
flex-nowrap flex-wrap: nowrap;
flex-wrap-rev flex-wrap: wrap-reverse;

Sets the alignment along the main-axis & how the available space will be distributed

Class Style
flex-main-start justify-content: flex-start;
flex-main-end justify-content: flex-end;
flex-main-center justify-content: flex-center;
flex-main-between justify-content: space-between;
flex-main-around justify-content: space-around;
flex-main-evenly justify-content: space-evenly;

Sets the alignment along the cross-axis - perpendicular to the main-axis

Class Style
flex-cross-start align-items: flex-start;
flex-cross-end align-items: flex-end;
flex-cross-center align-items: center;
flex-cross-stretch align-items: stretch;
flex-cross-baseline align-items: baseline;

Aligns multiple-line child-elements (items) when there is extra space in the cross-axis

Class Style
flex-extra-start align-content: flex-start;
flex-extra-end align-content: flex-end;
flex-extra-center align-content: flex-center;
flex-extra-stetch align-content: stretch;
flex-extra-between align-content: space-between;
flex-extra-around align-content: space-around;

flex-equal - Makes all children have equal width, along their main-axis.

- sets flex: 1; on all first-level Children, making them occupy equal space along the main axis.

flex-center - Centers the content both horizontally and vertically.

- sets both align-items: center; and justify-content: center; for all first-level Children, centering them along both the main-axis and the cross-axis.

flex-stretch-center - Stretches all children and centers their content on the main-axis.

1. Sets align-items: stretch; for all first-level Children, making them occupy all the cross-axis space available.

2. Sets display: flex; and align-items: center; on all first-level Children, centering their content along the cross-axis.

Classes applied on Child Elements follow the format : flex-self-{ value }. * These classes require the Parent Element to have the display: flex; in order to take effect.

Class Style
flex-self-start align-self: flex-start;
flex-self-end align-self: flex-end;
flex-self-center align-self: center;
flex-self-stretch align-self: stretch;
flex-self-baseline align-self: baseline;

To create different scenarios for specific viewports, any class can be prefixed with any *viewport*- or *viewport*-o- syntax.

Format :
<div class="{viewportPrefix}-{className}"> </div>
Example :
<div class="flex-row"> </div>
// sets flex-direction: row; on all viewports
Example :
<div class="sm-flex-row"> </div>
// sets flex-direction: row; on small(sm) viewports or larger
Example :
<div class="sm-o-flex-row"> </div>
// sets flex-direction: row; on small(sm) viewports only
