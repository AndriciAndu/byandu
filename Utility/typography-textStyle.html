Utility Classes : Text Style

Utility : Typography - Text Style

I. Description

A collection of utility classes, used to define an element's text styles.

Structured on 5 pillars: fontStyle | fontWeight | align | transform | decoration

II. Initialization

Download and refference the asterisk-utility-typography-textStyle.css CSS file.

<link href="--myPath--/asterisk-utility-typography-textStyle.css" type="text/css" rel="stylesheet">

Alternatively, a {*}-spec.css file can be generated within the [Code] Tab

<link href="--myPath--/asterisk-utility-typography-textStyle-spec.css" type="text/css" rel="stylesheet">

III. Usage

Classes are applied to the HTML Element and take effect on the HTML Element itself.

Shorthand Format :
<div class="text-{value}"> </div> Longhand  Format :
<div class="text-{propertyRefference}-{value}"> </div>
Example :
<div class="text-italic"> shorthand class </div>
// sets font-style: italic
Example :
<div class="text-fontStyle-italic"> longhand class </div>
// sets font-style: italic

IV. Available Classes

Each className follows the format : text-{ value } or text-{ propertyRefference }-{ value }.

* Classes which set styles have a both a longhand ex: text-fontStyle-bold and a shorthand ex: text-bold

* Classes which remove styles have only a longhand, because the values are shared across multiple classes:

ex: text-fonstStyle-inherit ex: text-fontWeight-inherit ex: text-align-inherit

V. Responsive Utilities : Viewport-Prefixes

To create different scenarios for specific viewports, any class can be prefixed with any *viewport*- or *viewport*-o- syntax.

* This applies for both shorthand and longhand classes.

Format :
<div class="{viewportPrefix}-{className}"> </div>
Example :
<div class="text-bold"> </div>
// sets font-weight: bold on all viewports
Example :
<div class="sm-text-bold"> </div>
// sets font-weight: bold on small(sm) viewports or larger
Example :
<div class="sm-o-text-bold"> </div>
// sets font-weight: bold on small(sm) viewports only
