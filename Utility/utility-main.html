I. Description

Simple responsive-wireframing and responsive-utility library.

Utility classes serve the purpose of handling specific styling/effect/responsiveness scenarios, which require only basic CSS rules, and for which would be redundant to create specific styling classes.

II. Initialization

Just download and refference the [ asterisk-utility.css ] file. * Prefferably done in the <head> seaction

<link href="--myPath--/asterisk-utility.css" type="text/css" rel="stylesheet">

III. Usage

The main Collection is formed of individual Collections, each covering a specific topic of interest.

Find out about each collection's specific details in their individual article (left navigation menu).

Each individual collection is formed of utility-specific-classes, which can be applied to any element in order to set/modify the refferenced attribute.

Ex: <div class="col-3"> width 25% </div> Ex: <div class="xs-hidden"> hidden on xs screen </div> Ex: <div class="p-0 m-0"> remove padding & margin </div>
