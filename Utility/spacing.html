Utility Classes : Spacing

Utility : Spacing

I. Description

Collection of , used to define an element's spacing.

Structured on 3 pillars: padding | margin | border

II. Initialization

Download and refference the asterisk-utility-spacing.css CSS file.

<link href="--myPath--/asterisk-utility-spacing.css" type="text/css" rel="stylesheet">

Alternatively, a {*}-spec.css file can be generated within the [Code] Tab

<link href="--myPath--/asterisk-utility-spacing-spec.css" type="text/css" rel="stylesheet">

III. Usage

Classes are applied to the HTML Element and take effect on the HTML Element itself.

Format :
<div class="{propertyRefference}-{size}"> </div>
Example :
<div class="m-0"> </div>
// sets margin: 0
Example :
<div class="pl-0"> </div>
// sets padding-left: 0
Example :
Example : <div class="bt-0"> </div>
// sets border-top: 0

All declarations have the !important syntax, in order to override other more-specific rules.

(when a spacing utility-class is used, it is usualy desired to have a definitive/overriding effect)

V. Responsive Utilities : Viewport-Prefixes

To create different scenarios for specific viewports, any class can be prefixed with any *viewport*- or *viewport*-o- syntax.

Format :
<div class="{viewportPrefix}-{className}"> </div>
Example :
<div class="m-0"> </div>
// sets margin: 0 on all viewports
Example :
<div class="sm-m-0"> </div>
// sets margin: 0 on small(sm) viewports or larger
Example :
<div class="sm-o-m-0"> </div>
// sets margin: 0 on small(sm) viewports only
