CSS Selectors

CSS Selector Syntax, which targets elements which have a class that starts with a specific string. 

									    [class^="classNameString"] , [class*=" classNameString"] {
									        /* do magic */
									    }

Example - both scenarios achieve the same results : 

								    imgFilter-1977  , imgFilter-aden    , imgFilter-amaro    ,
								    imgFilter-ashby , imgFilter-brannan , imgFilter-brooklyn {
								        /* do something */
								    }
								
								    [class^="imgFilter-"] , [class*=" imgFilter-"] {
								        /* do something */
								    }

CSS Selector Syntax, which targets elements where there are exactly x children/siblings.

From Lea Verou | CSS3 Secrets: 10 things you might not know about CSS3 | Fronteers 2011 . 

								    [mySelector]:first-child:nth-last-child(x),
								    [mySelector]:first-child:nth-last-child(x) ~ [mySelector] {
								        /* do magic */
								    }

CSS Selector Syntax, which targets elements where there are at least x children/siblings.

From Lea Verou | CSS3 Secrets: 10 things you might not know about CSS3 | Fronteers 2011 . 

								    [mySelector]:first-child:nth-last-child(n+x),
								    [mySelector]:first-child:nth-last-child(n+x) ~ [mySelector] {
								        /* do magic */
								    }
