Note: To function properly, hyphens require the lang attribute to be declared, representing the language used within the hyphenated area.

* This can be set on the <html> tag, if the language is the same throughout the whole document.

<html lang="en"> ... </html>

* This can also be set on any HTML Element within the document, if the language is different for its' contents.

Ex: <div lang="en"> Content in English </div>
Ex: <div lang="ro"> Continut in Romana </div>
