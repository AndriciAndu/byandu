Tooltip

Effects : Tooltip

I. Description

A collection of utility classes, which add a tooltip which appears when the Element is hovered.

* CSS-Only : The effect uses only HTML attributes and CSS classes - no Javascript involved.

* No Extra Markup : The effect is done by using the :before or :after pseudo-elements, without requiring extra HTML elements.

* No CSS Conflicts : The effect can be achieved by using a to hold the required classes - not affecting the HTML Element itself, in any way.

* Respnsive utilities : {direction} and {wrap} can be changed therough classes for specific viewports, through the {viewportPrefix}-{className} syntax.

* Supports line-breaks and horizontal-lines : Somewhat hacky, yet valid features.

II. Initialization

Download and refference the asterisk-effects-tooltip.css CSS file. CSS file.

<link href="--myPath--/asterisk-component-tooltip.css" type="text/css" rel="stylesheet">

Alternatively, a {*}-spec.css file can be generated within the [Code] Tab

<link href="--myPath--/asterisk-component-tooltip-spec.css" type="text/css" rel="stylesheet">

III. Usage

Stap 1. Add the tooltip-{pseudoElement} class on the HTML Element - defines which pseudo-element will be used

Stap 2. Add the tooltip-{direction} class on the HTML Element - defines the direction of the effect

Stap 3. Add the data-tooltipText attribute on the HTML Element - defines the text which will be shown on hover.

Format : <div class="tooltip-{pseudoElement} tooltip-{direction}" data-tooltipText="Text-to-be-shown"> </div>
Example : <div class="tooltip-before tooltip-left" data-tooltipText="Show me on hover!"> </div>

IV. Class-name Syntax

Class Type Values Description
tooltip-{direction} left | right | top | btm defines where the element will be positioned
tooltip-{pseudoElement} before | after defines which pseudo-element will be used
tooltip-{wrap} wrap | wrap-hyphen | wrap-each | nowrap defines if/how the text will flow to multiple lines

V. Styles

Word wrapping is enabled by default , allowing the text to flow onto multiple lines.

* This can be further customized by adding the tooltip-wrap-hyphen or tooltip-wrap-each class.

* The text can be set on a single line by adding the tooltip-nowrap class.

<div class="tooltip-before tooltip-top">
default / tooltip-wrap
<div class="tooltip-before tooltip-top tooltip-wrap-hyphen">
tooltip-wrap-hyphen
<div class="tooltip-before tooltip-top tooltip-wrap-each">
tooltip-wrap-each
<div class="tooltip-before tooltip-top tooltip-nowrap">
tooltip-nowrap

The tooltip's spacing uses borders with border-color set the same as background-color to simulate the tooltip's padding.

* This is required in order to enable adding <hr> -ish horizontal lines within the tooltip, without them being offset by paddings on one side, but not the other.

To set The tooltip's spacing (padding), modify the border-width value.

Available   Hacks    Features :

* These will work only if not using white-space: nowrap (if the text can actually flow to multiple lines)

* Adding line-breaks : insert &#13;&#10;

<div class="tooltip-before tooltip-top" data-tooltipText="1st line &#13;&#10; 2nd line &#13;&#10; 3rd line">
Hover me

* Adding horizontal lines : insert &#13;&#10;───────────&#13;&#10;

<div class="tooltip-before tooltip-top" data-tooltipText="1st line &#13;&#10;───────────&#13;&#10; 2nd line">
Hover me

* Add/Remove ─ characters to adjust the length of the line.

The distance-from-element sets the offset/distance of the tooltip from the HTML Element it is applied to.

It can be modified by changing the value of the --tooltip-distance CSS variable. The value must be a negative number.

:root {
--tooltip-distance: -1em;
}

VI. Responsive Utilities : Viewport-Prefixes

To create different scenarios for specific viewports, any tooltip-{direction} or tooltip-{wrap} class can be prefixed with any *viewport*- or *viewport*-o- syntax.

Format :
<div class="{viewportPrefix}-{className}"> </div>
Example :
<div class="tooltip-before tooltip-top"> </div> Example :
<div class="tooltip-before sm-tooltip-top"> </div> Example :
<div class="tooltip-before sm-o-tooltip-top"> </div>
Example :
<div class="tooltip-before tooltip-wrap"> </div> Example :
<div class="tooltip-before sm-tooltip-wrap"> </div> Example :
<div class="tooltip-before sm-o-tooltip-wrap"> </div>
