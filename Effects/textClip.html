I. Description

Collection of which apply a text-clip effect.

The effect results in clipping (cutting-out) the text and making it look as the first background available.

II. Initialization

Download and refference the [ asterisk-effects-textClip.css ] CSS file.

<link href="--myPath--/asterisk-effects-textClip.css" type="text/css" rel="stylesheet">

Alternatively, a {*}-spec.css file can be generated within the [Code] Tab

<link href="--myPath--/asterisk-effects-textClip-spec.css" type="text/css" rel="stylesheet">

III. Usage

The text-clip Class : Will hide the background entirely, leaving only the inner text 'visible'.

Step 1 : add the text-clip on the HTML Element.

Step 2 : add a background (image/gradient) on the HTML Element.

<div class="myDiv"> Without 'clip-text' class </div>

<div class="myDiv text-clip"> With 'clip-text' class </div>

.myDiv {

background: url('demo-img.jpg') center/cover no-repeat;

}

Without 'clip-text' class
With 'clip-text' class

The text-clip-{color} Classes : Will not hide the background, but only black / white are available as background-colors.

Step 1 : add the text-clip-{color} on the HTML Element.

Step 2 : add a background (image/gradient) on the Parent Element.

<div class="myParent">

<div class="text-clip-black"> My Clipped Text </div>

</div>

.myParent {

background: url('demo-img.jpg') center/cover no-repeat;

}
