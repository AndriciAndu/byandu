The 'Loading Spinner' Component

Components : Spinner

Simple spinner elements.

Step 1. Download and refference the asterisk-component-loadingSpinner.css CSS file.

<link href="--myPath--/asterisk-component-loadingSpinner.css" type="text/css" rel="stylesheet">

Alternatively, a {*}-spec.css file can be generated within the [Code] Tab

<link href="--myPath--/asterisk-component-loadingSpinner-spec.css" type="text/css" rel="stylesheet">

Step 1. Add the Component's markup to your HTML. * Recommended to be placed right after the <body> tag.

Step 2. Add a function to remove the spinner (once the Document has loaded).

* acts as a Container for the actual spinner

* allows for adapting to the size of the Element is is placed within, without modifying the size of the spinner

* uses flex properties to centers the spinner (horizontal and vertical)

* allocates the size of the actual spinner.

* holds the color property, which is inherited through currentColor, used to set the required color styling for the spinner

* changing it's width/height, will update the size of the actual loader/loader_elements.

* changing it's color, will update the scolor of the actual loader/loader_elements.

The loader_element elements are individual parts that make up the actual loader.

All spinner Elements are sized at width: 100px; height: 100px;

The ::before, ::after and/or .spinner__element Elements are dynamically sized, and will adapt to the changes of their spinner Parent Element.

* The width and height values must be equal, or the Elements will become deformed.

Width / Height : 100px
.spinner {
width: 100px; height: 100px;
}
