I. Description

Baseline Tabs component.

* Can be nested - Tabs Components can have other Tabs Components as Children, without creating conflicts or requiring additional modifications.

* Tab Navigation - within Tabs-item elements in their collapsed state, are not focusable by keyboard navigation.

Step 3. Activate or Delegate the Components' functionality:

asterisk.components.tabs.activate( tabs__HTMLElem )

A click eventListener will be added to the first .tabs-btnContainer descendant of the tabs__HTMLElem.

This delegates the functionality to the .tabs-btn Elements within said .tabs-btnContainer.

This will allow for only the tabs provided as an argument, to function.

asterisk.components.spoiler.delegate( target__HTMLElem )

A click eventListener will be added directly to the target__HTMLElem Element

This delegates the functionality to all .tabs-btn descendant Elements

This will allow for all tabs, which are descendants of the target__HTMLElem, to function.

III. Usage

Just add the Component's Markup to your Document :

* tabs-btn elements and tabs-content elements are synchronised with each other by having the same data-tabs-id value.

* each synchronised tabs-btn and tabs-content elements require an unique data-tabs-id value, within the tabs. 


										    <div class="tabs">
										        <div class="tabs-btnGroup">
										            <button class="tabs-btn tabs-btn--active"      data-tabs-id="1"> Button 1 </button>
										            <button class="tabs-btn"                       data-tabs-id="2"> Button 2 </button>
										            <button class="tabs-btn"                       data-tabs-id="3"> Button 3 </button>
										        </div>

										        <div class="tabs-contentGroup">
										            <div class="tabs-content tabs-content--active" data-tabs-id="1"> Content 1 </div>
										            <div class="tabs-content"                      data-tabs-id="2"> Content 2 </div>
										            <div class="tabs-content"                      data-tabs-id="3"> Content 3 </div>
										        </div>
										    </div>

Step 1. Add a new button with the tabs-btn class, within the tabs-btnGroup Container Element.

Step 2. Add a new div with the tabs-content class, within the tabs-contentGroup Container Element.

Step 3. Give the newly-added tabs-btn and tabs-content Elements the same data-tabs-id value.

* the data-tabs-id value must be unique within the current tabs Parent Element.

The tabs Component follows the same as all components within the asterisk library.

All the Functions are stored within the asterisk.components.tabs Object.

All can be accessed globally, through asterisk.components.tabs.{function}(). 

												    asterisk = {
												        components : { 
												            tabs : {       // This is the tabs object 
												                activate()    // Example of a tabs Function
												            }
												        } 
												    };

												    asterisk.components.tabs.activate()

The {*}.activatedItems Array is used for storing the Elements which have a tabs-related eventListener attached to them.

* It is updated automatically when an Element is initialized/terminated through the {*}.activate(), {*}.delegate() or {*}.terminate() methods.

Its' primary role is to be used within {*}.checkDOM(), which de-refferences Elements that have been removed from the DOM.

* {*}.checkDOM() is called automatically within asterisk.route.js's {*}.loadPage() method, creating a certain degree of automatic Memory Management. 

													    asterisk.components.tabs.activate = function( tabs__HTMLElem ) {
													        const target_tabsBtnGroup = tabs__HTMLElem.getElementsByClassName('tabs-btnGroup')[0];
													        target_tabsBtnGroup.addEventListener('click', asterisk.components.tabs.eventHandler_delegate);
													        asterisk.components.tabs.activatedItems.push(target_tabsBtnGroup);
														};

* Attaches a click eventListener to the first .tabs-btnGroup descendant of the tabs__HTMLElem, with the {*}.eventHandler_delegate() eventHandler.

The eventListener is not attached to each individual .tabs-btn Element, as it would be an waste of memory, regarding both the eventListener itself, and the on-element tabsParams.

The eventListener is not attached to the .tabs Element, as it would trigger multiple times for nested tabs.

* Adds the .tabs-btnGroup descendant to the {*}.activatedItems Array. 

													    asterisk.components.tabs.delegate = function( target__HTMLElem ) {
													        target__HTMLElem.addEventListener('click', asterisk.components.tabs.eventHandler_delegate);
													        asterisk.components.tabs.activatedItems.push(target__HTMLElem)
													    };

* Attaches a click eventListener to the target__HTMLElem, with the {*}.eventHandler_delegate() eventHandler.

-- This allows for all descendant .tabs-btn Elements to function, through Event Delegation.

* Adds the target__HTMLElem to the {*}.activatedItems Array. 

													    asterisk.components.tabs.terminate = function( target__HTMLElem ) {

													        const tabsObj = asterisk.components.tabs;
													        target__HTMLElem.removeEventListener('click', tabsObj.eventHandler_run);
													        target__HTMLElem.removeEventListener('click', tabsObj.eventHandler_delegate);
													        tabsObj.activatedItems = tabsObj.activatedItems.filter(item => item !== target__HTMLElem);

													        tabsObj.parameters_remove(target__HTMLElem);
													    };

* Removes the click eventListener (if attached with the {*}.activate() or {*}.delegate() methods), from the target__HTMLElem

* Removes the target__HTMLElem from the {*}.activatedItems Array.

* Removes the on-Element .spoilerParams Property from the target__HTMLElem. 

													    asterisk.components.tabs.eventHandler_run = function(e) {

													        const target = e.target;

													        const targetTabsBtnGroup = asterisk.utility.getParentByClassName(target, 'tabs-btnGroup');
													        const targetParams = targetTabsBtnGroup.tabsParams || asterisk.components.tabs.parameters_set(target)

													        // update [.tabs-btn] Elements
													        targetParams.target_buttons.map(btn => btn.classList.remove('tabs-btn--active'));
													        target.classList.add('tabs-btn--active');

													        // update [.tabs-content] Elements
													        const currentIndex = target.getAttribute('data-tabs-id');
													        targetParams.target_contentItems.map(function(item) {
													            if (item.getAttribute('data-tabs-id') == currentIndex) { 
													                item.classList.add   ('tabs-content--active') 
													            } else { 
													                item.classList.remove('tabs-content--active')
													            }
													        })

													    };

* Event Handler - This is the actual functionality of showing/hiding the .tabs-content.

It is never atually attached to any Element, when calling the {*}.activate(Element / {*}.delegate(Element), but called from within {*}.eventHandler_delegate.

Its' kept this way to follow the same arhitecture as all other asterisk components.

* Gets the tabsParams - or sets them up through {*}.parameters_set()

* Updates the state of the synchronised .tabs-btn and .tabs-content Elements.

													    asterisk.components.tabs.eventHandler_delegate = function(e) {
													        if ((" "+e.target.className+" ").indexOf(" tabs-btn ")>-1) { 
													            asterisk.components.tabs.eventHandler_run(e) 
													        }
													    };

* Event Handler used for delegating the functionality of the tabs Component.

It is attached to the first .tabs-BtnGroup descendant of the provided tabs-Element, when calling the {*}.delegate(tabs-Element).

It is attached to the provided non-Tabs-Element, when calling the {*}.delegate(non-Tabs-Element).

Its' existential purpose is that of a wrapper function, to allow the removal of the eventListener later on, through {*}.terminate(). 

													    asterisk.components.tabs.parameters_set = function( tabsBtnGroup__HTMLElem ) {

													        const target_contentGroup = asterisk.utility.getParentByClassName(tabsBtnGroup__HTMLElem, 'tabs').getElementsByClassName('tabs-contentGroup')[0];
													        const target_contentItems = Array.from(target_contentGroup.getElementsByClassName('tabs-content')).filter(function(item){
													            return item.parentElement === target_contentGroup
													        });

													        const target_buttons = Array.from(tabsBtnGroup__HTMLElem.getElementsByClassName('tabs-btn'));

													        tabsBtnGroup__HTMLElem.tabsParams = { target_contentItems , target_buttons };

													        return { target_contentItems , target_buttons }
													    };

* Defines the targets involved in the execution of the tabs Component's show/hide state change.

* Sets them as an on-element Property, on the .tabs-btnGroup Element, provided as the tabsBtnGroup__HTMLElem argument.

This allows for a faster execution of the function, as it removes the requirement to traverse the DOM and find the adequate targets on each itteration.

* It is used within {*}.eventHandler_run(), and it is automatically called when it fires for the first time.

This can also be done manually, from the start, to ensure a quicker first-time execution of the tabs Elements.

Example: Array.from(document.getElementsByClassName('tabs-btnGroup')).map(x => component__tabs.parameters_set(x)) 
													    asterisk.components.tabs.parameters_remove = function( target__HTMLElem ) {
													        if ((" "+target__HTMLElem.className+" ").indexOf(" tabs-btnGroup ")>-1) { 
													            target__HTMLElem.tabsParams = null
													        } else {
													            Array.from(target__HTMLElem.getElementsByClassName('tabs-btnGroup')).map(function(item){
													                item.tabsParams = null
													            });
													        }
													    };

* Removes the on-element spoilerParams Property from the Element provided as the target__HTMLElem argument, or from it's .spoiler-btn descendants.

It is used within {*}.terminate() 

													    asterisk.components.spoiler.checkDOM = function() {
													        const spoilerObj = asterisk.components.spoiler;
													        const targets = spoilerObj.activatedItems.filter(item => (item !== document) && !document.contains(item) );
													        targets.map(item => spoilerObj.terminate(item));
													    };

* Checks the items within the {*}.activatedItems Array.

* If any item is no longer in the DOM, remove the eventListeners and spoilerParams property, through {*}.terminate().

This de-refferences the Element, making it elligible for Garbage Collection.

* {*}.checkDOM() is called automatically within asterisk.route.js's {*}.loadPage() method, creating a certain degree of automatic Memory Management.

												        var asterisk = asterisk || {} ;

												    // Asterisk Component - [Tabs]
												    // -----------------------------------------------------

												        (function(){

												            // Component.Object Setup 
												            // -----------------------------------------------------
												            
												                asterisk.components.tabs = {};
												                var tabsObj = asterisk.components.tabs;

												            // Dependencies (not required if using [asterisk-utility.js])
												            // -----------------------------------------------------
												            
												                // hasClass()
												                // -----------------------------------------------------

												                    if (hasClass === undefined) {

												                        function hasClass( elem , klass ) {
												                            return (" " + elem.className + " " ).indexOf( " " + klass + " " ) > -1
												                        }

												                    };
												                
												                // getParent()
												                // -----------------------------------------------------

												                    if (getParent === undefined) {
												                        
												                        function getParent( elem, identifier ) {

												                            elemParent = elem.parentElement;

												                            if (elemParent) {

												                                var firstChar = identifier.charAt(0);

												                                if  (verify())  { return elemParent } 
												                                else            { return getParent(elemParent, identifier) }

												                            } else { return null };

												                            // ---
												                            function verify() {

												                                var identif_substr = identifier.substring(1); // remove first char - for class/id cases

												                                if      (firstChar === '.') { return ( " " + elemParent.className + " " ).indexOf( " " + identif_substr + " " ) > -1 } 
												                                else if (firstChar === '#') { return (elemParent.id      == identif_substr) } 
												                                else                        { return (elemParent.tagName == identifier.toUpperCase()) }
												                            }
												                        }

												                    };  

												            // Init
												            // -----------------------------------------------------

												                tabsObj.init = function(current_target, delegateEventListener) {
												                    
												                    if (current_target) {

												                        if (hasClass(current_target, 'tabs')) {

												                            tabsObj.activate(current_target)

												                        } else if (delegateEventListener === true) {

												                            tabsObj.delegate(current_target)

												                        } else { 

												                            var tabs = Array.from(current_target.getElementsByClassName('tabs'));   
												                            tabs.map(tabs => tabsObj.activate(tabs));

												                        }

												                    } else {

												                        var allTabs = Array.from(document.getElementsByClassName('tabs'));  
												                        allTabs.map(tabs => tabsObj.activate(tabs));    

												                    }

												                };

												            // Activate 
												            // -----------------------------------------------------

												                tabsObj.activate = function(current_target) {

												                    current_target = current_target.getElementsByClassName('tabs-btnGroup')[0];

												                    current_target.addEventListener('click', function(e) {
												                        if (hasClass(e.target, 'tabs-btn')) { tabsObj.run(e.target) }
												                    })
												                };

												            // Delegate
												            // -----------------------------------------------------

												                tabsObj.delegate = function(current_target) {

												                    current_target.addEventListener('click', function(e) {
												                        if (hasClass(e.target, 'tabs-btn')) { tabsObj.run(e.target) }
												                    })
												                };
												                    
												            // Run
												            // -----------------------------------------------------

												                tabsObj.run = function(current_tabBtn) {

												                    // update 'active' status on Btn
												                    var tabButtons = Array.from(current_tabBtn.parentElement.getElementsByClassName('tabs-btn'));
												                    tabButtons.map(btn => btn.classList.remove('tabs-btn--active'));
												                    current_tabBtn.classList.add('tabs-btn--active');

												                    // update 'expanded' status on content
												                    var currentIndex = current_tabBtn.getAttribute('data-tabs-id');
												                    var targetParent = getParent(current_tabBtn, '.tabs');

												                    var contentItems = Array.from(targetParent.getElementsByClassName('tabs-content'));
												                    contentItems = contentItems.filter(item => getParent(item, '.tabs') === targetParent);
												                    contentItems.map(function(item) { 
												                        if (item.getAttribute('data-tabs-id') == currentIndex) { item.classList.add   ('tabs-content--active') }
												                        else                                                    { item.classList.remove('tabs-content--active') }
												                    })

												                };
												            
												        })();
