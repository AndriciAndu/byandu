<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/font-awesome/4.7.0/css/font-awesome.min.css">

Step 2. Download and refference the asterisk-component-input.css CSS file.

<link href="--myPath--/asterisk-component-input.css" type="text/css" rel="stylesheet">

Baseline Markup

* With this markup, all styling options and effect are made available.

* The field__icon and field__validation Elements can be hidden (see below).

															
														    <label class="field--text">
														        <div class="field__icon"> <i class="fa fa-user" aria-hidden="true"></i> </div>
														        <input placeholder="Placeholder" required/>
														        <div class="field__validation"></div>
														    </label>

The Border Styles can be changed by adding any one of the field--border--rounded, field--border--round or field--border--btm Classes on the <label class="field--text"> Element.

The visible border is not applied on the <input> Element, but on the .field::before pseudo-element. This allows for :

* Dinamically adapting to the presence/absence of the .field__icon and field__validation Elements.

* Modifications to the border-size will not offset the position/size of the .field, or of the .field__icon / .field__validation icons. 

																		
																	    <label class="field--text"> ... </label>
																	    <label class="field--text field--border--rounded"> ... </label>
																	    <label class="field--text field--border--round"> ... </label>
																	    <label class="field--text field--border--btm"> ... </label>

The .field__validation Element is made visible only when the <input> Element has the required attribute.

Subsequently, the .field__validation Element can be hidden by :

1. Removing the <div class="field__validation"> from the Markup (not reccomended)

2. Removing the required attribute on the Element.

3. Adding the field--validation--hidden Class on the label> Element. 

																		
																	    <label class="field--text">
																	        <div class="field__icon"> <i class="fa fa-user" aria-hidden="true"></i> </div>
																	        <input placeholder="Placeholder" required/>
																	        <div class="field__validation"></div>
																	    </label>

The .field__icon contains a Font Awesome Icon, which can be changed directly within the Element.

It can be hidden by :

1. Removing the required attribute on the Element.

2. Adding the field--icon--hidden Class on the label> Element.
