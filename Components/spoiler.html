Component : Spoiler

Components : Spoiler

Baseline Spoiler component.

* Free placement - All Spoiler Buttons and Spoilers do not have co-dependent proximity, and can be placed anywhere within the document.

* Linked Buttons - Multiple Spoiler Buttons can point to the same Spoiler, and have their state linked between each other.

* Tab Navigation - within Spoilers in their collapsed state, Elements are not focusable by keyboard navigation.

* Precise Animation - max-height is calculated based on actual content height and spacing.

* Unobstructive Functionality - Function will fail silently and console.log details, if no adequate Spoiler is found (no match for data-spoiler-id value).

* Auto-adaptable for dynamic content - Will automatically adjust it's height for Dynamic content (expanding/collapsing menus or adding/removing HTML Elements).

Step 1. Add the Component's Markup to your Document :

* spoiler elements and spoiler-btn elements can be placed anywhere within the document, regardless of each other's position.

Step 2. Set a unique data-spoiler-id value.

* spoiler and spoiler-btn elements are synchronised with each other by having the same data-spoiler-id value.

												    <button class="spoiler-btn" data-spoiler-id="mySpoilerId-1"> </button>

												    <div class="spoiler" data-spoiler-id="mySpoilerId-1">
												        <div class="spoiler-content">
												             <!-- content goes here --> 
												        </div>
												    </div>

spoiler-btn elements can be given the spoiler-btn-autoText class which adds an 'Expand' / 'Collapse' string, after the innerText provided.

<button class="spoiler-btn" data-spoiler-id=""> My Text </button>
<button class="spoiler-btn spoiler-btn-autoText" data-spoiler-id=""> </button>
<button class="spoiler-btn spoiler-btn-autoText" data-spoiler-id=""> My Text </button>

* Acts as a wrapper for the spoiler-content Element, in order to prevent the spoiler-content Element's box-model styles from offsetting the collapsed state (paddings / borders).

* The expanded/collapsed state max-height value change will be applied to it.

* Must not have paddings / borders.

* Holds all of the spoiler's content.

* It can have margins / paddings / borders, without affecting the collapsed state.

* can be placed anywhere within the document, while still targetting the adequate spoiler.

* multiple spoiler-btn Elements can point to the same spoiler Element, by having the same data-spoiler-id value.

* Multiple spoiler-btn Elements that are pointing to the same spoiler Element, have their state synchronised.

The Component has two States : collapsed and expanded

The State is changed when the user clicks a spoiler-btn element.

Each State is primarily reflected by the presence/absence of a specific class on the spoiler and spoiler-btn Elements. 

												    // Collapsed 
												    <div class="spoiler-btn"> My Button </div>
												    <div class="spoiler"> ... </div>
												
												    // Expanded 
												    <div class="spoiler-btn spoiler-btn--active"> My Button </div>
												    <div class="spoiler spoiler--expanded"> ... </div>

The spoiler Component follows the same as all components within the asterisk library.

All the Functions are stored within the asterisk.components.spoiler Object.

All can be accessed globally, through asterisk.components.spoiler.{function}(). 

												    asterisk = {
												        components : { 
												            spoiler : {       // This is the spoiler object 
												                activate()    // Example of a spoiler Function
												            }
												        } 
												    };

												    asterisk.components.spoiler.activate()

The {*}.activatedItems Array is used for storing the Elements which have a spoiler-related eventListener attached to them.

* It is updated automatically when an Element is initialized/terminated through the {*}.activate(), {*}.delegate() or {*}.terminate() methods.

Its' primary role is to be used within {*}.checkDOM(), which de-refferences Elements that have been removed from the DOM.

* {*}.checkDOM() is called automatically within asterisk.route.js's {*}.loadPage() method, creating a certain degree of automatic Memory Management. 

													    asterisk.components.spoiler.activate = function( spoilerBtn__HTMLElem ) {
													        spoilerBtn__HTMLElem.addEventListener('click', asterisk.components.spoiler.eventHandler_run);
													        asterisk.components.spoiler.activatedItems.push(spoilerBtn__HTMLElem)
													    };

* Attaches a click eventListener to the spoilerBtn__HTMLElem, with the {*}.eventHandler_run() eventHandler.

* Adds the target__HTMLElem to the {*}.activatedItems Array. 

													    asterisk.components.spoiler.delegate = function( target__HTMLElem ) {
													        target__HTMLElem.addEventListener('click', asterisk.components.spoiler.eventHandler_delegate);
													        asterisk.components.spoiler.activatedItems.push(target__HTMLElem)
													    };

* Attaches a click eventListener to the target__HTMLElem, with the {*}.eventHandler_delegate() eventHandler.

-- This allows for all descendant .spoiler-btn Elements to function, through Event Delegation.

* Adds the target__HTMLElem to the {*}.activatedItems Array. 

													    asterisk.components.spoiler.terminate = function( target__HTMLElem ) {

													        const spoilerObj = asterisk.components.spoiler;
													        target__HTMLElem.removeEventListener('click', spoilerObj.eventHandler_run);
													        target__HTMLElem.removeEventListener('click', spoilerObj.eventHandler_delegate);
													        spoilerObj.activatedItems = spoilerObj.activatedItems.filter(item => item !== target__HTMLElem);

													        spoilerObj.parameters_remove(target__HTMLElem);
													    };

* Removes the click eventListener (if attached with the {*}.activate() or {*}.delegate() methods), from the target__HTMLElem

* Removes the target__HTMLElem from the {*}.activatedItems Array.

* Removes the on-Element .spoilerParams Property from the target__HTMLElem. 

													    asterisk.components.spoiler.eventHandler_run = function(e) {

													        const spoilerParams = e.target.spoilerParams || asterisk.components.spoiler.parameters_set(e.target , true);
													        const targetSpoiler = spoilerParams.targetSpoiler;
													        const targetButtons = spoilerParams.targetButtons;

													        targetSpoiler.style.maxHeight = targetSpoiler.scrollHeight + 'px';

													        if (hasClass(targetSpoiler , 'spoiler--expanded')) { // collapsing animation
													            setTimeout(function(){ targetSpoiler.style.maxHeight = 0 }, 34);
													            targetButtons.map((btn) => btn.classList.remove('spoiler-btn--active'));
													        } else {
													            targetButtons.map((btn) => btn.classList.add('spoiler-btn--active'));
													        };

													        targetSpoiler.classList.toggle('spoiler--expanded');
													    };

* Event Handler - This is the actual functionality of collapsing/expanding the .spoiler.

It is attached to the provided Element, when calling the {*}.activate(Element).

Its' existential purpose is that of a wrapper function, to allow the removal of the eventListener later on, through {*}.terminate().

* Gets the spoilerParams - or sets them up through {*}.parameters_set()

* Updates the state of the synchronised .spoiler-btn and .spoiler Elements.

													    asterisk.components.spoiler.eventHandler_delegate = function(e) {
													        if ((" "+e.target.className+" ").indexOf(" spoiler-btn ")>-1) { 
													            asterisk.components.spoiler.eventHandler_run(e) 
													        }
													    };

* Event Handler used for delegating the functionality of the spoiler Component, to Elements that are not .spoiler-btn elements.

It is attached to the provided Element, when calling the {*}.delegate(Element).

Its' existential purpose is that of a wrapper function, to allow the removal of the eventListener later on, through {*}.terminate(). 

													    asterisk.components.spoiler.parameters_set = function( spoilerBtn__HTMLElem ) {

													        const targetId = spoilerBtn__HTMLElem.getAttribute('data-spoiler-id');

													        const targetSpoiler = Array.from(document.getElementsByClassName('spoiler')).find(function(spoiler) { 
													            return spoiler.getAttribute('data-spoiler-id') === targetId 
													        });
													        const targetButtons = Array.from(document.getElementsByClassName('spoiler-btn')).filter(function(spoilerBtn) { 
													            return spoilerBtn.getAttribute('data-spoiler-id') === targetId 
													        });

													        spoilerBtn__HTMLElem.spoilerParams = { targetSpoiler , targetButtons };

													        if (targetSpoiler) {
													            return { targetSpoiler , targetButtons }
													        } else {
													            console.log(
													                'No [spoiler] element with [data-spoiler-id="'+targetId+'"] found within document.' + 
													                ' Event triggered by [spoiler-btn] : ' , spoilerBtn__HTMLElem );
													        }
													    };

* Defines the targets involved in the execution of the spoiler Component's collapse/expand state change.

* Sets them as an on-element Property, on the .spoiler-btn Element, provided as the spoilerBtn__HTMLElem argument.

This allows for a faster execution of the function, as it removes the requirement to traverse the DOM and find the adequate targets on each itteration.

* It is used within {*}.eventHandler_run(), and it is automatically called when it fires for the first time.

This can also be done manually, from the start, to ensure a quicker first-time execution of the spoiler Elements.

Example: Array.from(document.getElementsByClassName('spoiler-btn')).map(x => component__spoiler.parameters_set(x)) 
													    asterisk.components.spoiler.parameters_remove = function( target__HTMLElem ) {
													        if ((" "+target__HTMLElem.className+" ").indexOf(" spoiler-btn ")>-1) { 
													            target__HTMLElem.spoilerParams = null
													        } else {
													            Array.from(target__HTMLElem.getElementsByClassName('spoiler-btn')).map(function(item){
													                item.spoilerParams = null
													            });
													        }
													    };

* Removes the on-element spoilerParams Property from the Element provided as the target__HTMLElem argument, or from it's .spoiler-btn descendants.

It is used within {*}.terminate() 

													    asterisk.components.spoiler.checkDOM = function() {
													        const spoilerObj = asterisk.components.spoiler;
													        const targets = spoilerObj.activatedItems.filter(item => (item !== document) && !document.contains(item) );
													        targets.map(item => spoilerObj.terminate(item));
													    };

* Checks the items within the {*}.activatedItems Array.

* If any item is no longer in the DOM, remove the eventListeners and spoilerParams property, through {*}.terminate().

This de-refferences the Element, making it elligible for Garbage Collection.

* {*}.checkDOM() is called automatically within asterisk.route.js's {*}.loadPage() method, creating a certain degree of automatic Memory Management.

												            // -----------------------------------------------------

												                asterisk.components.spoiler = {};
												                var spoilerObj = asterisk.components.spoiler;

												            // Dependencies (not required if using [asterisk-utility.js])
												            // -----------------------------------------------------

												                function hasClass( elem , klass ) {
												                    return (" " + elem.className + " " ).indexOf( " " + klass + " " ) > -1
												                }

												            // Init
												            // -----------------------------------------------------

												                spoilerObj.init = function(current_target, delegateEvent__boolean) {

												                    if (current_target) { 

												                        if (hasClass(current_target, 'spoiler-btn')) {

												                            spoilerObj.activate(current_target) 

												                        } else if (delegateEvent__boolean === true) {

												                            spoilerObj.delegate(current_target)

												                        } else { 

												                            var spoilerBtns = Array.from(current_target.getElementsByClassName('spoiler-btn')); 
												                            spoilerBtns.map(spoilerBtn => spoilerObj.activate(spoilerBtn));

												                        }

												                    } else {

												                        var allSpoilerBtns = Array.from(document.getElementsByClassName('spoiler-btn'));
												                        allSpoilerBtns.map(spoilerBtn => spoilerObj.activate(spoilerBtn));

												                    }

												                };

												            // Activate 
												            // -----------------------------------------------------

												                spoilerObj.activate = function(current_target) {

												                    current_target.addEventListener('click', function() { 
												                        spoilerObj.run(this) 
												                    })

												                };

												            // Delegate
												            // -----------------------------------------------------

												                spoilerObj.delegate = function(current_target) {

												                    current_target.addEventListener('click', function(e) {
												                        if (hasClass(e.target, 'spoiler-btn')) { spoilerObj.run(e.target) }
												                    })

												                };
												                    
												            // Run
												            // -----------------------------------------------------

												                spoilerObj.run = function(current_spoilerBtn) {

												                    if (!current_spoilerBtn.targetParameters) { 
												                        spoilerObj.spoilerBtn_defineParameters(current_spoilerBtn)
												                    };

												                    var targetSpoiler = current_spoilerBtn.targetParameters.targetSpoiler;
												                    var targetButtons = current_spoilerBtn.targetParameters.targetButtons;

												                    if (targetSpoiler) {

												                        if (hasClass(targetSpoiler , 'spoiler-expand')) {

												                            setTimeout(function(){ targetSpoiler.style.maxHeight = 0 }, 0);
												                            
												                            targetSpoiler.classList.remove('spoiler-expand');
												                            targetButtons.map((btn) => btn.classList.remove('active'));

												                        } else {

												                            var spoiler_content = targetSpoiler.getElementsByClassName('spoiler-content')[0];

												                            var margin_top = parseInt(getComputedStyle(spoiler_content, null).getPropertyValue("margin-top"));
												                            var margin_btm = parseInt(getComputedStyle(spoiler_content, null).getPropertyValue("margin-bottom"));
												                            var content_height = spoiler_content.offsetHeight;

												                            var maxHeight_toSet = content_height + margin_top + margin_btm;

												                            setTimeout(function(){ targetSpoiler.style.maxHeight = maxHeight_toSet + 'px' }, 0);

												                            targetSpoiler.classList.add('spoiler-expand');                  // update  spoiler  status
												                            targetButtons.map((btn) => btn.classList.add('active'));        // update  button   status
												                        }

												                    } else {

												                        console.log('No [spoiler] element with [data-spoiler-id="'+current_spoilerBtn.getAttribute('data-spoiler-id')+'"] found within document. Event triggered by [spoiler-btn] :');
												                        console.log(current_spoilerBtn);

												                    }

												                };

												            // Define Parameters
												            // -----------------------------------------------------
												            // Performance - skip checking document (for target-elements : spoiler | spoiler-btn) on each function call

												                // Will set the target parameters for each [spoiler-btn]
												                // will run automatically on first spoiler-btn click
												                // will be more efficient for future calls there-after

												                spoilerObj.spoilerBtn_defineParameters = function(current_spoilerBtn) {

												                    if (!current_spoilerBtn.targetParameters) {

												                        current_spoilerBtn.targetParameters = {};

												                        var targetId = current_spoilerBtn.getAttribute('data-spoiler-id');

												                        var spoilers = Array.from(document.getElementsByClassName('spoiler'));
												                        current_spoilerBtn.targetParameters.targetSpoiler = spoilers.filter(spoiler => spoiler.getAttribute('data-spoiler-id') === targetId )[0];

												                        var spoilerButtons = Array.from(document.getElementsByClassName('spoiler-btn'));
												                        current_spoilerBtn.targetParameters.targetButtons = spoilerButtons.filter(spoilerBtn => spoilerBtn.getAttribute('data-spoiler-id') === targetId );
												                
												                    }

												                };

												            // Run initial parameter-definition (optional || not recommended if dynamically-generated content)
												            // Best called as one of the final functions in the scripts 
												            // -----------------------------------------------------

												                spoilerObj.spoilerBtn_defineParameters_all = function() {

												                    var spoilerButtons = Array.from(document.getElementsByClassName('spoiler-btn'));
												                    spoilerButtons.map(btn => asterisk.components.spoiler.spoilerBtn_defineParameters(btn));

												                };

												        })();
